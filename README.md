# Toy Robot

### Requirements

- Node v12 or higher
- Yarn 1.21.1 or higher

### Tools Use

- Github Actions
- Node
- Yarn
- Npx Code Coverage

### Unit Tests

- Run `yarn install`
- Run `yarn unit`

### Code Coverage

- Run `yarn install`
- Run `yarn coverage`

### CLI

- Run `yarn start`
- Enter the possible commands: 
  - PLACE X,Y,F
  - MOVE
  - LEFT
  - RIGHT
  - REPORT
